(매칭)
A : 매칭된 목격 게시글 B : 매칭된 실종 게시글 
    -> A입장 헤더에 : B게시글
       B입장 헤더에 : A게시글, 위치업로드버튼

(목격게시글 통한 채팅)
A : 목격 게시글 B : 해당 목격 게시글 하단 버튼으로 진입
—> A입장 헤더에 : 게시글 안 나타남.
   B입장 헤더에 : A게시글

(실종게시글 통한 채팅)
A : 실종 게시글 B : 해당 실종 게시글 하단 발견 카드 제보 통해 채팅방 진입
-> A입장 헤더에 : 게시글 안 나타남, 위치 업로드 버튼, 발견 카드 
B입장 헤더에 : 발견 카드 

위치 업로드 버튼은 상단 고정이고 발견카드는 채팅방 내용 중 첫번째로 들어감
